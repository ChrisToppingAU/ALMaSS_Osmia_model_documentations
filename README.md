# ALMaSS_Osmia_model_documentations
Holds the ALMaSS *Osmia bicornis* model code and MIDox documentation

## Overview 
This repository contains the MIDox (Model Implementation Documentation with Doxygen) for the *Osmia bicornis* solitary bee model within the ALMaSS framework.
